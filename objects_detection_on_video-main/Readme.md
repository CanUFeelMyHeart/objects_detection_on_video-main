## Как работать?
1. Склонировать репозиторий на компьютер с поиощью команды: `git clone`
2. Установить зависимости через команды:
`run libraries.bat` или
`pip install -r requirements.txt`
3. Изменить путь к видео на строчке 368: `video = "C:\\Users\\your_path"`
4. Запустить скрипт: `python main.py`
